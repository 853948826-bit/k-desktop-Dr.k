\begin{abstract}
This paper presents $\kappa$-Desktop, a desktop automation agent based on the Three-Box Intelligence architecture. The system dynamically routes tasks through three pathways (fast, medium, slow) based on an information precision index ($\kappa$) that grows with successful executions. We demonstrate phase transitions between the three phases (gas, liquid, crystal) as $\kappa$ increases.
\end{abstract}

\section{Introduction}
Desktop automation systems often face a trade-off between reliability and intelligence. $\kappa$-Desktop addresses this with a three-box architecture that adapts to task complexity and historical success rates.

\section{Three-Box Architecture}
The system uses three pathways:
\begin{itemize}
    \item \textbf{Fast Path} ($\kappa > 0.8$): Crystal phase - direct execution of learned patterns
    \item \textbf{Medium Path} ($0.3 < \kappa < 0.8$): Liquid phase - rule-based reasoning
    \item \textbf{Slow Path} ($\kappa < 0.3$): Gas phase - LLM-based generation
\end{itemize}

\section{Information Precision Index ($\kappa$)}
The $\kappa$ value follows the growth curve:
\[
\kappa(n) = 0.95(1 - e^{-0.25n}) + 0.05
\]
where $n$ is the number of successful executions.
